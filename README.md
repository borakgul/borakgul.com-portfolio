## Getting Started

1. **Clone the repository:**
    ```bash
    git clone https://github.com/borakgul/borakgul.com-portfolio.git
    ```
2. **Navigate to the project directory:**
    ```bash
    cd borakgul.com-portfolio
    ```
3. **Install dependencies:**
    ```bash
    npm install
    ```
4. **Start the development server:**
    ```bash
    npm run dev
    ```
